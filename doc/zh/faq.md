# 常见问题

## 我需要先删除之前的安装吗？

不需要。安装程序会自动更新 `~/.opencode` 中的所需目录。

## 我可以不使用 API 密钥吗？

可以。无需密钥的集成可以直接使用。需要密钥的集成如果未设置密钥则会被禁用。

## 安装后如何更改智能体的响应语言？

最简单的方法是重新运行安装程序并设置新的"首选响应语言"值。

## 我可以为不同的智能体选择不同的模型吗？

可以。在安装过程中有一个步骤"选择要设置显式模型的智能体（可选）"，您可以为选定的智能体设置模型。

## 将密钥存储在 `~/.opencode/keys` 中安全吗？

这是本地用户密钥存储。请勿发布这些文件，也请勿将它们提交到 git。

## 如果智能体行为在更新后发生变化怎么办？

1. 检查您的 `agents/skills` 中的手动编辑是否被覆盖。
2. 验证已启用的 MCP 集成。
3. 如需要，从 zip 备份中恢复已知良好的状态。
