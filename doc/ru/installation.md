# Установка

## Требования

- Node.js `>= 20.12.0`
- Доступ к интерактивному терминалу (TTY)

Проверка версии Node.js:

```bash
node -v
```

## Команда установки

```bash
npx -g @ozerohax/assistagents@latest
```

Эта же команда используется и для обновления до последней версии.

## Что происходит во время установки

Установщик пошагово спрашивает:

1. Нужно ли создать zip-бэкап текущего `~/.opencode`.
2. Предпочитаемый язык ответов агентов.
3. Нужно ли явно задать модели для отдельных агентов (опционально).
4. Нужны ли экспериментальные hash-based file tools.
5. Какие MCP-интеграции включить.
6. Значения ключей только для тех интеграций, где они требуются.

## Какие пути обновляются

В процессе запуска установщик перезаписывает/создает:

- `~/.opencode/agents/`
- `~/.opencode/skills/`
- `~/.opencode/commands/`
- `~/.opencode/tools/` (если включены hash-инструменты)
- `~/.opencode/keys/`
- `~/.opencode/opencode.jsonc`

## Что важно перед запуском

- Если в `~/.opencode/agents`, `~/.opencode/skills` и `~/.opencode/commands` есть ваши правки, включите backup.
- Если запускаете в CI или в неинтерактивной оболочке, установка завершится ошибкой из-за отсутствия TTY.
