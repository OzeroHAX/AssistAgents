## Коротко о пакете

`assistagents` - это интерактивный установщик, который подготавливает каталог `~/.opencode`:

- устанавливает/обновляет `agents`, `skills` и `commands` из шаблонов пакета;
- опционально устанавливает экспериментальные hash-инструменты в `tools`;
- помогает включить MCP-интеграции;
- сохраняет ключи в `~/.opencode/keys`;
- генерирует `~/.opencode/opencode.jsonc` с нужными MCP-конфигурациями и ограничениями по permission.

## Важные ограничения

- Установщик работает только в интерактивном TTY-терминале.
- Повторный запуск заменяет `~/.opencode/agents`, `~/.opencode/skills` и `~/.opencode/commands`.
- Если у вас были ручные правки этих директорий, включайте backup перед обновлением.
