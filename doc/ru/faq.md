# FAQ

## Нужно ли удалять старую установку перед запуском?

Нет. Установщик сам обновляет нужные каталоги в `~/.opencode`.

## Можно ли использовать пакет без API-ключей?

Да. Интеграции без ключей доступны сразу. Интеграции с обязательными ключами будут отключены, если ключ не задан.

## Где изменить язык ответов агентов после установки?

Проще всего повторно запустить установщик и указать новый `Preferred response language`.

## Можно ли выбрать разные модели для разных агентов?

Да. Во время установки есть шаг `Select agents to set explicit models (optional)`, где можно задать модель отдельно для каждого выбранного агента.

## Безопасно ли хранить ключи в `~/.opencode/keys`?

Это локальное хранилище ключей пользователя. Не публикуйте эти файлы и не добавляйте их в git.

## Что делать, если после обновления поведение агентов изменилось?

1. Проверьте, не перезаписались ли ваши ручные правки в `agents/skills`.
2. Сверьте список включенных MCP-интеграций.
3. При необходимости восстановите рабочее состояние из zip-бэкапа.
