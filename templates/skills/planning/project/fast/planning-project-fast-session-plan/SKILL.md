---
name: planning-project-fast-session-plan
description: Timeboxed session plan with deliverables and checkpoints to maintain momentum and prevent scope creep.
---

<skill_overview>
  <purpose>Split work into 3-4 short sessions, each ending with a verifiable outcome.</purpose>
  <triggers>
    <trigger>User asks for an "evenings" plan or a hard time limit</trigger>
    <trigger>You need to reduce dropout risk and make progress inevitable</trigger>
  </triggers>
</skill_overview>

<rules>
  <rule>Each session must end with a working artifact you can show.</rule>
  <rule>Each session must have a yes/no checkpoint and acceptance criteria.</rule>
  <rule>End each session with a short "what we will NOT do next" list (to prevent drift).</rule>
</rules>

<session_template>
  <field>Goal (1 line)</field>
  <field>Objective (what will work at the end)</field>
  <field>Steps (3-6 steps with rough time estimates)</field>
  <field>Checkpoint (how to verify success)</field>
  <field>Out of scope for this session (1-3 bullets)</field>
</session_template>

<first_15_minutes>
  <instruction>Include a "First 15 minutes" block with 3 concrete actions so the user can start immediately.</instruction>
</first_15_minutes>
